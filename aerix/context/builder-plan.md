# builder-rules.md

## Purpose

Defines behavior for the page builder.

---

# Layout

Desktop:

- Sidebar
- Editor
- Preview

Mobile:

- Tabs

Tabs:

- Blocks
- Editor
- Preview

---

# Block Order

Supported Blocks:

- Profile
- Social Icons
- Contact Button
- Intro Video
- YouTube
- Instagram
- Links
- Gallery
- Spotify
- Text
- Divider

---

# Default Blocks

New User:

- Profile
- Contact Button
- Links

---

# Drag And Drop

Use:

Framer Motion

Users can:

- Reorder Blocks
- Rearrange Sections

Preview updates immediately.

---

# Visibility

Blocks can be:

- Visible
- Hidden

Hidden blocks:

Remain saved.

Not rendered publicly.

---

# Profile Block

Required

Contains:

- Profile Image
- Banner
- Name
- Username
- Bio

Cannot be deleted.

---

# Contact Button

Enabled by default.

Can be disabled.

---

# Intro Video

Optional

Supports:

- Upload
- URL

Limits:

- 30 Seconds
- 25 MB

---

# YouTube

Supports:

Multiple videos.

Single video:

Static layout.

Multiple videos:

Carousel.

---

# Instagram

Supports:

Multiple reels.

Displayed as horizontal scroll.

---

# Links

Core feature.

At least one link recommended.

Supports:

- Title
- URL
- Icon

---

# Gallery

Supports:

- Images
- Videos

Displayed as horizontal scrolling gallery.

---

# Spotify

Optional block.

Accepts Spotify URLs.

---

# Preview

Preview must match public page exactly.

No preview-only UI.

No fake rendering.

---

# Save Behavior

Changes autosave.

Manual publish button available.

---

# Principle

Builder should feel simple.

Users should not need technical knowledge.
