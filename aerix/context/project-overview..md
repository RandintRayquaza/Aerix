# project-overview.md

## What is Aerix?

Aerix is a creator identity platform that allows users to create a personalized public profile page.

The platform combines:

* Social Links
* Media
* Videos
* Branding
* Analytics

inside a single shareable page.

Users can publish pages using:

aerix.live/username

or

aerix.live/custom-slug

---

## Authentication

The product now includes dedicated login, signup, and onboarding screens under the App Router auth route group.

Landing page CTAs should route into:

* /login
* /signup
* /onboarding

---

## Core Features

### Profile

Users can configure:

* Profile Image
* Banner Image
* Name
* Username
* Bio

---

### Social Links

Users can showcase:

* Instagram
* YouTube
* X
* LinkedIn
* Website
* Discord
* Reddit
* Spotify
* Custom Links

---

### Intro Video

Optional creator introduction video.

Supports:

* Upload
* External URL

---

### Featured YouTube Videos

Embedded YouTube videos.

Multiple videos become a carousel.

Single videos remain static.

---

### Instagram Reels

Embedded Instagram reels.

Displayed in horizontal scroll layout.

---

### Gallery

Users can share:

* Images
* Short Videos

through uploads or URLs.

---

### Themes

Theme customization system including:

* Colors
* Backgrounds
* Presets
* Creator themes

---

### Analytics

Track audience engagement and traffic performance.

---

### Media Library

Centralized location for:

* Images
* Videos
* URLs

used throughout the platform.

---

### Domain Management

Manage:

* Username URL
* Custom slug

without external domains.

---

## Product Philosophy

Aerix focuses on:

* Simplicity
* Performance
* Mobile-first usage
* Creator branding
* Easy customization

The goal is to give creators a professional online identity without requiring technical skills.
