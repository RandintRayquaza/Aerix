# media-rules.md

## Purpose

Defines media upload behavior.

---

# Storage Provider

ImageKit

Used For:

- Images
- Videos
- CDN Delivery

---

# Storage Limit

Per Account:

500 MB

Includes:

- Banner
- Profile Image
- Gallery
- Videos
- Intro Video

---

# Supported Media

Images

- Upload
- URL

Videos

- Upload
- URL

---

# Image Formats

Supported:

- JPG
- JPEG
- PNG
- WEBP

---

# Video Formats

Supported:

- MP4
- MOV
- WEBM

Convert to MP4 when needed.

---

# Intro Video

Maximum:

- 30 Seconds
- 25 MB

---

# Gallery Videos

Maximum:

- 30 Seconds
- 25 MB

---

# Duplicate Media

If identical media already exists:

Reuse existing URL.

Do not duplicate storage.

---

# Deletion

Deleting media:

- Removes reference
- Updates storage usage
- Updates previews

---

# External URLs

Supported:

Images

Videos

Store URL only.

Do not duplicate assets.

---

# Preview

Media previews should load instantly.

Use optimized ImageKit transformations.

---

# Upload Validation

Validate:

- File Type
- File Size
- Duration

before upload.

Reject invalid files immediately.

---

# Future Expansion

Reserved For:

- Premium Storage
- Media Collections
- Subscriber Content

Not included in v1.
